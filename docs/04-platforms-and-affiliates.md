### 플랫폼 상세

#### X (트위터) ⭐⭐⭐⭐⭐
```
장점:
- 제휴 링크 제한 없음
- API 무료 티어 제공 (읽기/쓰기)
- 해시태그로 발견 가능성
- 리트윗으로 확산

단점:
- 같은 링크 반복 → 스팸 처리
- 글자 수 제한 (280자)
- 최근 API 정책 변경 잦음

자동화:
- Twitter API v2 (무료: 1,500 트윗/월)
- 유료: $100/월 (Basic)

정책 주의:
- 자동화 계정 명시 권장
- 동일 내용 반복 금지
- URL 단축 서비스 주의 (차단될 수 있음)
```

#### 스레드 (Threads) ⭐⭐⭐⭐
```
장점:
- 인스타와 연동
- 링크 자유롭게 가능
- 새로운 플랫폼 (경쟁 적음)
- 텍스트 중심

단점:
- API 아직 제한적
- 사용자층 아직 적음
- 검색/발견 기능 약함

자동화:
- 공식 API 2024년 출시 (제한적)
- 비공식 방법은 위험

정책 주의:
- 메타 정책 따름
- 스팸 판정 시 인스타도 영향
```

#### 페이스북 페이지 ⭐⭐
```
장점:
- 거대한 사용자 기반
- 그룹 기능 활용 가능

단점:
- 외부 링크 도달률 1-5%
- 제휴 링크 포함 시 더 하락
- 자동화 매우 어려움
- 광고 집행 없으면 노출 안됨

자동화:
- Graph API (심사 필요)
- 페이지 자동 포스팅은 승인 어려움

정책 주의:
- "광고성 콘텐츠" 판정 시 제재
- 외부 링크 알고리즘 패널티
- 제휴 링크는 "광고" 표시 필수
```

#### 인스타그램 ⭐⭐
```
장점:
- 시각적 콘텐츠에 강함
- 스토리 링크 (팔로워 무관)
- 쇼핑 기능

단점:
- 피드에 링크 불가
- 자동화 시 계정 정지 위험 높음
- 프로필 링크 1개만

자동화:
- 공식 API는 비즈니스 기능만
- 비공식 도구 사용 시 밴 위험 매우 높음

정책 주의:
- 자동화 도구 엄격 금지
- 제휴 마케팅 명시 필수
- 스토리 링크 남용 시 제재

우회 방법:
- 링크트리, 링크인바이오 활용
- 스토리에 제품 태그 (쇼핑 기능)
```

#### 틱톡 ⭐⭐⭐
```
장점:
- 폭발적 도달 가능성
- 젊은 층 타겟
- 숏폼 영상 트렌드

단점:
- 링크는 프로필만
- 자동화 매우 어려움
- 영상 제작 필요

자동화:
- 공식 API 제한적
- 자동 업로드 어려움

정책 주의:
- 광고 표시 필수
- 금융/건강 제품 제한
- TikTok Shop 활용 권장
```

#### 핀터레스트 ⭐⭐⭐⭐
```
장점:
- 제휴 링크 친화적
- 이미지 기반 검색
- 장기 트래픽 (핀이 오래 살아있음)
- API 제공

단점:
- 국내 사용자 적음
- 특정 카테고리에 한정 (패션, 인테리어 등)

자동화:
- Pinterest API (무료)
- 예약 포스팅 가능

정책 주의:
- 제휴 링크 허용
- 하지만 "클로킹" 금지
- 광고 표시 권장
```

#### 마스토돈/블루스카이 ⭐⭐⭐
```
장점:
- 탈중앙화, 정책 자유로움
- API 완전 개방
- 기술 친화적 사용자

단점:
- 사용자 수 적음
- 마케팅 효과 미지수

자동화:
- API 완전 자유
- 봇 운영 환영하는 문화
```

## 2. 메신저/채팅 플랫폼

| 플랫폼 | 제휴 링크 | 자동화 | 난이도 | 정책 리스크 | 도달률 | 추천도 |
|--------|:--------:|:------:|:------:|:----------:|:------:|:------:|
| **텔레그램 채널** | ✅ 완전 자유 | ✅ Bot API | 매우 쉬움 | 매우 낮음 | 매우 높음 | ⭐⭐⭐⭐⭐ |
| **디스코드** | ✅ 가능 | ✅ Bot API | 쉬움 | 낮음 | 높음 | ⭐⭐⭐⭐ |
| **카카오톡 채널** | ⚠️ 제한적 | ⚠️ 심사 필요 | 중간 | 중간 | 높음 | ⭐⭐⭐ |
| **라인** | ⚠️ 제한적 | ⚠️ 심사 필요 | 중간 | 중간 | 높음 | ⭐⭐⭐ |
| **슬랙** | ✅ 가능 | ✅ API | 쉬움 | 낮음 | 높음 | ⭐⭐⭐ |
| **왓츠앱** | ⚠️ 제한적 | ❌ 어려움 | 어려움 | 높음 | 높음 | ⭐⭐ |

### 플랫폼 상세

#### 텔레그램 채널 ⭐⭐⭐⭐⭐
```
장점:
- 제한 없음 (링크, 파일, 길이)
- Bot API 완전 무료
- 도달률 거의 100%
- 채널/그룹 구분 가능
- 익명성

단점:
- 국내 사용자 상대적으로 적음
- 초기 구독자 확보 어려움
- "핫딜" 채널 경쟁 심함

자동화:
- python-telegram-bot (완전 무료)
- 무제한 메시지 발송
- 미디어 그룹, 버튼 등 다양한 기능

정책 주의:
- 거의 없음
- 불법 콘텐츠만 금지
```

#### 디스코드 ⭐⭐⭐⭐
```
장점:
- 커뮤니티 친화적
- 봇 API 무료, 강력
- 역할/채널 세분화 가능

단점:
- 게이머 중심 이미지
- 일반인 접근성 낮음

자동화:
- discord.py (무료)
- 봇 호스팅 필요 (무료 옵션 있음)

정책 주의:
- 스팸 서버 제재
- DM 스팸 엄격 금지
```

#### 카카오톡 채널 ⭐⭐⭐
```
장점:
- 국내 사용자 최다
- 알림톡/친구톡

단점:
- 비용 발생 (메시지당 과금)
- 심사 필요
- 스팸 판정 시 강력 제재

자동화:
- 카카오 비즈니스 API
- 알림톡: 건당 7-15원
- 친구톡: 건당 15-20원

정책 주의:
- 광고성 메시지 제한
- 야간 발송 제한
- 수신 동의 필수
```

| **워드프레스** | ✅ 완전 자유 | ✅ API | 중간 | 없음 | 매우 좋음 | ⭐⭐⭐⭐⭐ |

| **GitHub Pages** | ✅ 완전 자유 | ✅ Actions | 중간 | 없음 | 중간 | ⭐⭐⭐⭐ |

### 플랫폼 상세

#### 티스토리 ⭐⭐⭐⭐⭐
```
장점:
- 제휴 링크 허용
- 애드센스 병행 가능
- API 제공 (자동 포스팅)
- 카카오 검색 노출
- 무료

단점:
- 네이버 검색 노출 어려움
- 디자인 제한적

자동화:
- Tistory API (OAuth 2.0)
- 글 작성, 수정, 삭제 가능

정책 주의:
- 저품질 글 다량 생산 시 검색 제외
- 자동 생성 콘텐츠 주의
```

#### 네이버 블로그 ⭐⭐⭐
```
장점:
- 네이버 검색 노출 유리
- 국내 최대 트래픽

단점:
- 외부 제휴 링크 제한적
- 쿠팡 링크 → 저품질 판정 위험
- 자동화 불가
- 애드센스 불가

자동화:
- 공식 API 없음
- 비공식 방법은 계정 정지 위험

정책 주의:
- 제휴 링크 과다 시 저품질
- "체험단", "협찬" 표시 필수
- 네이버 인플루언서로 전환 권장
```

#### 워드프레스 (Self-hosted) ⭐⭐⭐⭐⭐
```
장점:
- 완전한 자유
- SEO 최적화 가능
- 플러그인 생태계
- 제휴 링크 무제한

단점:
- 호스팅 비용 ($3-10/월)
- 초기 설정 필요
- 보안 관리 필요

자동화:
- REST API (완전 개방)
- WP-CLI
- 다양한 자동화 플러그인

정책 주의:
- 없음 (자체 서버)
```

#### Substack ⭐⭐⭐⭐
```
장점:
- 뉴스레터 중심
- 유료 구독 기능
- 제휴 링크 허용
- SEO 괜찮음

단점:
- 뉴스레터 형식에 한정
- 디자인 제한

자동화:
- API 제한적
- 이메일 자동화는 가능
```

### 플랫폼 상세

#### 유튜브 ⭐⭐⭐⭐
```
장점:
- 설명란 링크 무제한
- 장기 트래픽 (검색)
- 광고 수익 + 제휴 수익 동시
- 신뢰도 높음

단점:
- 영상 제작 필요
- 초기 구독자 확보 어려움
- 얼굴/음성 노출 부담

자동화:
- YouTube Data API
- 업로드 자동화 가능
- 하지만 콘텐츠 생성은 수동

정책 주의:
- 유료 프로모션 체크박스 필수
- 제휴 링크 명시 권장
- 반복/저품질 콘텐츠 제재
```
