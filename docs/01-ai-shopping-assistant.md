# AI 쇼핑 어시스턴트

## 개요
사용자가 원하는 상품을 질문하면 여러 쇼핑몰의 가격과 제휴 링크를 자동으로 제공하는 챗봇 서비스

## 서비스 흐름
```
사용자 질문 → 상품 검색 → 가격 수집 → 제휴 링크 생성 → 응답 제공
     ↓              ↓            ↓              ↓
"에어팟 프로"   쿠팡/11번가   가격 비교      내 링크로 변환
                아마존 등
```

---

## 핵심 기능

### 1. 자연어 상품 검색
- 사용자가 "에어팟 프로 최저가", "20만원대 무선 이어폰 추천" 등 자연어로 질문
- AI가 의도를 파악하여 적절한 상품 검색

### 2. 멀티 플랫폼 가격 비교
| 플랫폼 | API 지원 | 크롤링 난이도 | 비고 |
|--------|----------|--------------|------|
| 쿠팡 | 파트너스 API | - | 공식 지원 |
| 11번가 | 오픈 API | - | 공식 지원 |
| 네이버 쇼핑 | 검색 API | - | 공식 지원 |
| G마켓/옥션 | 제한적 | 중간 | 스크래핑 필요 |
| 아마존 | PA-API | - | 승인 필요 |

### 3. 제휴 링크 자동 변환
- 각 플랫폼별 제휴 링크 형식으로 자동 변환
- 쿠팡: `https://link.coupang.com/re/AFFILIATE?...`
- 11번가: `https://www.11st.co.kr/...?affiliate=...`

### 4. 개인화 추천
- 사용자 검색 이력 기반 추천
- 선호 가격대, 브랜드 학습

## 기술 스택 옵션

### 챗봇 플랫폼
| 플랫폼 | 장점 | 단점 | 개발 난이도 |
|--------|------|------|------------|
| **텔레그램 봇** | API 제약 없음, 무료 | 국내 사용자 적음 | 쉬움 |
| **카카오톡 채널** | 국내 사용자 많음 | API 제한, 심사 필요 | 중간 |
| **디스코드 봇** | 커뮤니티 친화적 | 일반 사용자 접근성 낮음 | 쉬움 |
| **웹 채팅** | 접근성 좋음 | 서버 비용 발생 | 중간 |

### AI/NLP 엔진
| 옵션 | 비용 | 성능 | 특징 |
|------|------|------|------|
| **OpenAI GPT** | 유료 (API) | 높음 | 자연어 이해 뛰어남 |
| **Claude API** | 유료 (API) | 높음 | 긴 컨텍스트 처리 |
| **로컬 LLM (Ollama)** | 무료 | 중간 | 서버 리소스 필요 |
| **규칙 기반** | 무료 | 낮음 | 단순하지만 제한적 |

### 백엔드
| 옵션 | 언어 | 특징 |
|------|------|------|
| **FastAPI** | Python | 비동기, 타입 힌트, 문서 자동생성 |
| **Express.js** | Node.js | 생태계 풍부, 빠른 개발 |
| **Bun + Hono** | TypeScript | 빠른 성능, 현대적 |

### 데이터베이스
| 용도 | 추천 |
|------|------|
| 사용자 정보 | PostgreSQL, SQLite |
| 상품 캐시 | Redis |
| 검색 이력 | MongoDB |

## 수익 모델

### 직접 수익
- 제휴 링크 클릭 → 구매 시 수수료 (1-10%)
- 예상 전환율: 1-3%

### 수익 시뮬레이션
```
일일 사용자 100명
  → 클릭률 30% = 30클릭
  → 구매 전환율 3% = 0.9건
  → 평균 구매가 50,000원, 수수료 3%
  → 일 수익 = 0.9 × 50,000 × 0.03 = 1,350원
  → 월 수익 = 약 40,500원

일일 사용자 1,000명 시
  → 월 수익 = 약 405,000원
```
