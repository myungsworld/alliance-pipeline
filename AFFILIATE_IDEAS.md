# 제휴 마케팅 자동화 아이디어 정리

## 수익 구조 (기본 원리)

```
[나] → 제휴 링크 생성 → [사용자 클릭] → [구매] → [수수료 지급]
```

---

## 사용 가능한 제휴 프로그램

### 국내 쇼핑몰
| 플랫폼 | 수수료 | 특징 |
|--------|--------|------|
| 쿠팡 파트너스 | 1-3% | 가장 대중적, 24시간 쿠키 |
| 11번가 어필리에이트 | 1-2% | 신뢰도 높음 |
| 네이버 인플루언서 | 검색 노출 보상 | 블로그 연동 필수 |
| G마켓/옥션 | 1-2% | 이베이코리아 |
| SSG닷컴 | 1-2% | 신세계 계열 |

### 해외 쇼핑몰
| 플랫폼 | 수수료 | 특징 |
|--------|--------|------|
| 아마존 어소시에이츠 | 1-10% | 글로벌 최대 |
| 알리익스프레스 | 3-9% | 저가 상품 |
| iHerb | 5-10% | 건강식품 특화 |

### CPA/CPS 네트워크 (통합 플랫폼)
| 플랫폼 | 특징 |
|--------|------|
| 링크프라이스 | 국내 최대, 다양한 광고주 |
| 애드픽 | SNS 특화, 초보자 친화적 |
| 텐핑 | 모바일 최적화 |
| 아이라이크클릭 | 금융/보험 강점 |
| 리더스CPA | 고단가 캠페인 |

## 자동화 시스템 아이디어

### 1. AI 쇼핑 어시스턴트
```
사용자: "에어팟 프로 최저가 알려줘"

봇: 쿠팡 289,000원 (내 링크)
    11번가 285,000원 (내 링크)
    아마존 $199 (내 링크)
```
- 챗봇 형태로 상품 추천
- 모든 플랫폼 링크 통합
- 사용자 취향 학습

### 2. 가격비교 큐레이션 봇
- 여러 쇼핑몰 가격 자동 비교
- 최저가 알림 기능
- 할인 정보 자동 수집

### 3. 할인 정보 자동 알림 서비스
```
[데이터 수집] → [AI 필터링] → [채널 발행]
                              ├─ 텔레그램
                              ├─ 카카오톡 채널
                              ├─ 블로그
                              └─ 인스타/틱톡
```

---

## 배포 채널 옵션

| 채널 | 자동화 난이도 | 장점 |
|------|--------------|------|
| 텔레그램 | 쉬움 | API 제약 거의 없음 |
| 블로그 (티스토리) | 중간 | SEO 강함, 애드센스 가능 |
| 카카오톡 채널 | 중간 | 한국 사용자 접근성 |
| 인스타그램 | 어려움 | 젊은 층 타겟 |
| 틱톡 | 어려움 | 바이럴 가능성 |

---

## 빠른 시작 추천 조합

```
텔레그램 채널 + 블로그(티스토리) + 링크프라이스/쿠팡 파트너스
```

**이유:**
- 텔레그램: API 자유로움, 자동화 쉬움
- 티스토리: SEO 좋음, 장기 트래픽
- 링크프라이스: 여러 쇼핑몰 한 번에 연동
